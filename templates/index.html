Getting Started

  1. Go to Settings and add your accounts
  2. Configure auto-update interval
  3. Click "Analyze All" to start tracking
  4. View your statistics in the Dashboard
  5. Use the API to generate badges for your repositories

API Usage

Get Statistics:

GET /api/stats?account_id=1&language=python&period=week

Get Badge:

GET /api/badge/total_lines?language=python&color=%2308C

Available badge types:

  • total_lines
  • code_lines
  • comment_lines
  • empty_lines
  • files
